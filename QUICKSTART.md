# 快速开始指南

## 🚀 5分钟快速上手

### 步骤1：注册账号（30秒）
1. 打开应用，自动跳转到登录页
2. 点击"注册"按钮
3. 输入用户名（例如：admin）
4. 输入密码并确认
5. 点击"注册"按钮
6. ✨ 恭喜！您已成为管理员（第一个用户）

### 步骤2：添加查询网址（1分钟）
1. 登录后进入主页面
2. 在"网址管理"卡片中：
   - 输入网址：`https://example.com/green-cert`
   - 输入名称：`示例交易平台`
3. 点击"添加网址"
4. 网址出现在下方列表中

### 步骤3：查看示例数据（1分钟）
1. 在"数据查询"卡片中
2. 点击"执行查询"按钮
3. 等待查询完成
4. 在"交易数据"表格中查看结果

### 步骤4：使用日期筛选（1分钟）
1. 点击"开始日期"选择器
2. 选择今天的日期（2026-01-26）
3. 点击"执行查询"
4. 查看当日的交易信息

### 步骤5：查看详细信息（30秒）
1. 在交易数据表格中
2. 找到"详情链接"列
3. 点击链接图标
4. 在新标签页中打开原始页面

## 🔧 定制数据抓取

### 为什么需要定制？
当前应用提供了完整的数据抓取框架，但由于不同网站的HTML结构不同，需要根据实际目标网站编写解析逻辑。

### 如何定制？
1. **找到文件**：`supabase/functions/scrape-data/index.ts`
2. **定位函数**：`parseHtmlData()`
3. **分析网站**：使用浏览器开发者工具查看HTML结构
4. **编写代码**：根据HTML结构提取数据
5. **测试验证**：添加测试网址，执行查询验证结果

### 需要提取的字段
```typescript
{
  project_name: '项目名称',        // 必填
  bidding_unit: '招标单位',       // 可选
  bidder_unit: '投标单位',        // 可选
  winning_unit: '中标单位',       // 可选
  total_price: 15000000.00,      // 数字
  unit_price: 150.00,            // 数字
  detail_link: 'https://...',    // URL
  is_channel: true,              // 布尔值
  cert_year: 2026,               // 整数
  transaction_date: '2026-01-26' // 日期
}
```

## ❓ 常见问题

**Q: 点击查询后没有数据？**
A: 这是正常的。当前Edge Function提供的是框架代码，需要根据实际网站HTML结构编写解析逻辑。示例数据已经预置在数据库中，可以直接查看。

**Q: 如何查看示例数据？**
A: 登录后，在主页面直接查看"交易数据"表格，或者点击"执行查询"按钮刷新数据。

**Q: 可以删除示例数据吗？**
A: 可以。示例数据仅用于演示，实际使用时会被真实数据替换。

**Q: 忘记密码怎么办？**
A: 当前版本不支持密码找回。建议记住密码或使用密码管理工具。
