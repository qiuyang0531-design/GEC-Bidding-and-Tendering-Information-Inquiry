# 中国绿色电力证书交易查询应用需求文档

## 1. 应用概述

### 1.1 应用名称
中国绿色电力证书交易查询应用

### 1.2 应用描述
本应用用于查询中国绿色电力证书交易的投标、中标信息，支持用户添加需要查询的网址，通过自动抓取方式获取交易数据，并提供按日期筛选和查询功能，方便用户快速获取当日的交易信息。\n
## 2. 核心功能

### 2.1 用户系统
- 用户注册：支持用户通过账号密码进行注册
- 用户登录：支持用户通过账号密码进行登录

### 2.2 网址管理
- 网址添加：用户可直接添加需要查询的网址
- 网址列表：展示用户已添加的所有网址

### 2.3 数据查询与展示
- 自动抓取：系统自动抓取用户添加的网址中的交易数据
- 查询功能：提供查询按钮，用户点击后执行数据查询
- 日期筛选：支持按日期筛选交易记录
- 数据展示：查询结果在单页中集中展示，显示以下内容：\n  - 项目名称
  - 招标/投标/中标单位信息
  - 对应的总价
  - 绿证单价
  - 具体标的信息获取的链接
  - 绿证是通道/非通道
  - 绿证年份

### 2.4 数据存储
- 数据保存：查询到的交易数据保存到数据库中

## 3. 页面结构

### 3.1 登录页面
- 账号输入框
- 密码输入框
- 登录按钮
- 注册按钮

### 3.2 注册页面
- 账号输入框
- 密码输入框
- 确认密码输入框
- 注册按钮

### 3.3 主页面
- 网址添加区域：输入网址、添加按钮
- 网址列表：已添加的网址列表
- 日期筛选器：日期选择控件
- 查询按钮：执行数据查询
- 数据展示区域：以列表或表格形式展示查询结果

## 4. 技术要求

### 4.1 数据抓取
- 需要实现网页抓取功能，从用户添加的网址中提取交易相关信息
- 支持处理不同格式的网页数据

### 4.2 数据存储
- 需要数据库支持，用于保存抓取到的交易数据
- 数据库需要包含字段：项目名称、单位信息、总价、单价、链接、通道类型、年份、日期等

### 4.3 数据展示
- 查询结果需在单页中集中展示，支持按日期筛选
- 数据展示需清晰明了，方便用户快速查看交易信息
